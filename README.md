# my-ayx-notes

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容多样，不针对任何特定网站或域名，主要作为个人笔记和页面资料的集中存放地。

## 目录说明

```
my-ayx-notes/
├── index.html          # 主入口页面（可选）
├── pages/              # 存放所有独立 HTML 页面
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
├── assets/             # 公共资源文件（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

## 页面归档说明

- 每个 HTML 页面均为独立完整文件，可单独打开使用。
- 页面内容可能包括工具页面、资料整理、学习笔记等，不限于单一主题。
- 页面之间无强依赖关系，部分页面可能引用 `assets/` 目录下的公共资源。

## 维护说明

- 本仓库不定期更新，新增或修改页面会直接提交到 `main` 分支。
- 页面命名尽量保持清晰，避免使用无意义缩写。
- 如需引用本仓库中的页面，请直接使用原始文件链接。
- 欢迎通过 Issues 提交建议或问题，但请勿要求添加特定内容或推广链接。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/my-ayx-notes.git
   ```
2. 直接在浏览器打开 `pages/` 目录下的任意 HTML 文件即可查看内容。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后在线访问。
